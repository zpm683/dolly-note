# DollyNote

DollyNote 是一个只有单个 HTML 文件的极简白板。

你可以用它绘图、写字、平移、缩放、把当前画布状态保存到 URL hash 中，并导出一个可再次打开和继续编辑的独立 HTML 文件。导出的文件像是当前白板的一个“克隆体”，这也是 DollyNote 这个名字的来源。

## 功能特性

- 单文件应用，不需要构建流程
- 支持平移和缩放的无限感画布
- 提供画笔、文本、橡皮擦工具
- 提供多种颜色和线宽
- 支持撤销与重做
- 可将当前白板状态保存进 URL hash
- 可将当前白板导出为独立 HTML 文件
- 导出的 HTML 重新打开后仍可继续编辑和再次导出

## 工作原理

DollyNote 会把当前绘图内容和视口状态编码进页面的 hash。

点击保存按钮后，应用会把当前白板序列化成紧凑的 hash 字符串，并写入地址栏 URL。

点击下载按钮后，DollyNote 会基于当前页面生成一个新的 HTML 文件，把最新 hash 作为启动状态注入进去，再将该文件下载到本地。打开导出的 HTML 后，白板内容会自动恢复。

## 使用方式

1. 在浏览器中打开 [index.html](index.html)。
2. 使用画笔绘图、插入文字，或者擦除内容。
3. 通过拖拽、滚轮或触控手势在画布中移动。
4. 点击保存，把包含当前状态的分享链接写入 URL 并复制出去。
5. 点击下载，导出当前白板的独立 HTML 副本。

## 快捷键与操作

- `P`：画笔
- `T`：文本
- `E`：橡皮擦
- `0`：重置视图
- `Ctrl/Cmd + Z`：撤销
- `Ctrl/Cmd + Y`：重做
- `Ctrl/Cmd + Shift + Z`：重做
- `Space` + 拖拽：平移
- 鼠标滚轮 / 触控板：平移
- `Ctrl` 或 `Cmd` + 滚轮：缩放

## 项目结构

- [index.html](index.html)：完整应用入口，包含界面、渲染、状态编码、恢复逻辑，以及独立 HTML 导出能力

## 说明

- 当前项目依赖 Google Fonts 和 LZ-String CDN。
- 导出的 HTML 文件会保留继续编辑的能力。
- 导出文件名会带上时间戳，避免意外覆盖。
